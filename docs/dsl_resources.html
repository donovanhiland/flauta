(static) resources(name, optionsopt) → {Array.<Route>}

Creates 5 endpoints for working with the a given resource: + GET /api/v1/{name} (handler = index) + GET /api/v1/{name}/:id (handler = show) + POST /api/v1/{name} (handler = create) + PATCH /api/v1/{name}/:id (handler = update) + DELETE /api/v1/{name}/:id (handler = destroy)
