(static) propOrLazy(prop, thunk, target) → {*}

Takes a property to return (if present) and a thunk to evaluate if not. If the target object has the property it is returned. If the target object does not have the property then the thunk is invoked and it's return value is returned instead.

(static) tapLog(tag, loggeropt, …data) → {*}

Fulfills a common use-case which is the desire to log out data flowing through a pipe at a specific point in the pipeline with a specific tag value. tapLog taps into an R.pipe or R.compose to log out the given data at that stage of the pipe/composition. See R.tap documentation for more information.
Parameters:
Name Type Attributes Description
tag string The tag value to log out before the given data.
logger function <optional>
 The logger to use (default is console.log).
data * <repeatable>
 Any set of data you wish to log.
